RESEAU DISTANT (WAN)
Configuration du WAN :
Pour configurer l’interface WAN, veuillez entrer le numéro du PVC ATM approprié ou cocher l’option suivante afin d’activer le service WAN sur l’interface eth0. Veuillez noter qu’avant de sélectionner cette option vous devez supprimer toutes les configurations de PVC ATM existantes.
 
De même, lorsque vous choisissez cette option, vous ne pourrez plus configurer d’autres PVC ATM jusqu’à ce que le service WAN sur Ethernet soit supprimé.
 
 
  
CONFIGURATION DU PVC ATM :
Ce menu vous permet de modifier les paramètres de votre PVC ATM.
 
ATTENTION :
La modification de ces paramètres peut entraîner l'inactivité de votre service ADSL.
Il est conseillé de ne pas changer les valeurs ci-dessous à moins d'y être invité par un technicien de Club Internet.
 
VPI: (0-255)
VCI: (32-65535)
 
Catégorie de service:
 
Taux maximal de cellules [cells/s]
Taux cellulaire soutenable [cells/s]
Taille maximal de rafale [cells]
 
ACTIVER LA QUALITÉ DE SERVICE (QOS):
 
Pour autoriser la gestion de la qualité de service et améliorer la performance des classes d'applications, veuillez cocher par défaut la case ci-dessous.
